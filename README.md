# 3d-Lake-Simulator
Kalpit Kothari
Yasasvi V Peruvemba
### Instructions to run :

```shell
make run
```

### Technologies used :

* C++
* Glut (OpenGL)
* Textures

### Features :

* Terrains
* Textures
* Interactive Lake
* Skybox
* Time of the day :
  * Sky
* Full camera control with mouse and keyboard.
* Text rendering (Instructions and time)
* Collision avoidance
* Ripple Creation
* Reflection on Water Surface
