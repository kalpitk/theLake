# 3d-map-simulator

### Instructions to run :

```shell
make run
```

### Technologies used :

* C++
* Glut (OpenGL)
* Textures

### Features :

* Terrains
* Textures
* Skybox
* Time of the day :
  * Sky
* Full camera control with mouse and keyboard.
* Text rendering (Instructions and time)
* Collision avoidance
